# Hello World 示例运行指南

## 快速开始

每个示例都展示了从简单到复杂的实现方式，您可以根据需要选择合适的版本进行学习和实验。

## 运行环境要求

### JavaScript/Node.js
```bash
# 检查 Node.js 版本
node -v
npm -v

# 创建项目目录
mkdir hello-project
cd hello-project
npm init -y
```

### Python
```bash
# 检查 Python 版本
python3 --version

# 创建虚拟环境（推荐）
python3 -m venv venv
source venv/bin/activate
```

### Java
```bash
# 检查 Java 版本
java -version
javac -version
```

### Rust
```bash
# 检查 Rust 版本
rustc --version
cargo --version
```

## 逐步运行示例

### 1. 运行基础示例

#### JavaScript
```bash
# 直接运行
node js-basic.js

# 输出：
# Hello, World!
# Hello, World!
# Hello, World!
# 同步: Hello immediately
# 异步: Hello after 1 second
```

#### Python
```bash
# 直接运行
python3 py-basic.py

# 输出：
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
```

#### Java
```bash
# 编译
javac java-basic.java

# 运行
java HelloWorld

# 输出：
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
```

#### Rust
```bash
# 创建项目
cargo new hello_rust
cd hello_rust

# 复制基础代码到 src/main.rs
# 运行
cargo run

# 输出：
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
# Hello, World!
```

### 2. 运行 Web 服务器示例

#### Express.js
```bash
# 初始化项目并安装依赖
npm init -y
npm install express

# 运行
node js-express.js

# 访问：
# http://localhost:3000
# http://localhost:3000/greet/John
```

#### Flask
```bash
# 安装依赖
pip install flask

# 运行
python3 py-flask.py

# 访问：
# http://localhost:5000
# http://localhost:5000/greet?name=John
# http://localhost:5000/api/greet
```

#### FastAPI
```bash
# 安装依赖
pip install fastapi uvicorn

# 运行
python3 py-fastapi.py

# 或使用 uvicorn
uvicorn py-fastapi:app --reload

# 访问：
# http://localhost:8000
# http://localhost:8000/docs (API 文档)
# http://localhost:8000/greet/John
```

### 3. 高级示例

#### React 组件
```bash
# 创建 React 应用
npx create-react-app react-hello
cd react-hello

# 替换 src/App.js 内容
npm start
```

#### Spring Boot
```bash
# 使用 Spring Initializr 创建项目
# 访问：https://start.spring.io/

# 添加依赖：
# - Spring Web
# - Spring Boot DevTools

# 运行主类或使用 Maven/Gradle
mvn spring-boot:run
```

#### Axum Web 服务器
```bash
# 创建项目
cargo new axum_hello
cd axum_hello

# 添加依赖到 Cargo.toml
[dependencies]
axum = "0.6"
tokio = { version = "1.0", features = ["full"] }
serde = { version = "1.0", features = ["derive"] }

# 运行
cargo run

# 访问：
# http://localhost:3000
# http://localhost:3000/greet/John
```

## 调试和扩展建议

### 1. 添加日志
在每个示例中添加更多的日志输出，了解程序的执行流程。

### 2. 修改参数
修改输入参数，观察输出结果的变化。

### 3. 错误处理
尝试添加错误处理逻辑，了解如何处理异常情况。

### 4. 性能测试
对于 Web 服务器示例，可以使用 ab 或 wrk 进行性能测试。

```bash
# Apache Bench 测试
ab -n 1000 -c 10 http://localhost:3000/

# Wrk 测试
wrk -t12 -c400 -d30s http://localhost:3000/
```

## 常见问题

### Q: 为什么选择不同的语言？
A: 每种语言都有其特定的优势和适用场景。了解它们的特点有助于选择合适的技术栈。

### Q: 如何选择学习路径？
A: 根据您的兴趣和职业目标选择。前端开发侧重 JavaScript，后端可以选择多种语言，数据科学首选 Python。

### Q: 如何深入学习？
A: 阅读官方文档、参与开源项目、解决实际问题，这是最好的学习方式。
